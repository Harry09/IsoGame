## Gamedev rules
http://gynvael.coldwind.pl/?id=657

## Controls
- W/S/A/D - move
- Arrows - shot

### Compile in debug to
- R - Open all portals in current room
- T - Close all portals in current room
- G - Generate new map
- X - Show all rooms
- Q - invincible player
- F - Epic stats
